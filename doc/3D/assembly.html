Building standoff-hinged Walking Linkage, 3D printed

This drawing will help explain my conventions for naming parts of the linkage:

Models to print

you can, of course, print all parts before starting assembly. However, in the assembly instructions we recommend which parts to print just-in-time. We recommend that you assemble the chassis in this manner, so that if there is a problem with a part, it can be re-printed as needed. It also allows you to build the chassis while it is printing minimizing total project time.

Assembly

Recommended assembly order:
  1. Print and glue parts that need to be glued.
    1. Print pulleys and crank arms (AC).
      (2) main pulleys, mainPulley.stl
      (4) crank arms, AC.stl
    2. Print (4) main bars, mainBar.stl
    3. Print (2) motor mounts, motorMount.stl
    4. Glue motor mounts to main bars
    5. Glue standoffs to main pulley (for each side)
    6. Assemble main pulley into main bar (twice, once for each side)
    7. Glue crankarms (AC) onto main pulley standoffs.
      Make sure they are glued/locked at level 180 degree phase.
    8. Print (8) hips, (4) hip.stl, and (4) hip.stl after a mirror reflection
    9. Glue forks onto Hips.
  2. Print rest of one pair of leg assemblies. You should have already printed and glued all 8 hip parts, 4 of each of the 2 mirror symmetries. Each leg pair will use two of the above hips, one in each of the mirror symmetries. In addition to the two hips, one of each mirror pair, you will need:
    • (4) EF links, EF.stl
    • (4) BH links, BH.stl
    • (2) CH links, CH.stl
    • (2) CD links, CD.stl
    • (2) feet, foot.stl, one of each mirror symmetry

    Clean support material from foot/hamstring node.
    Polish 3/16" hole surfaces.
  3. Construct a pair of legs, check for smoothness of operation.
  4. Once operation of leg pair has been verified, print and assemble another leg pair
  5. Complete assembly of one side of legs (4 legs on a main bar, with motor drive), and mount on axles
  6. Test motor operation with a small battery.
  7. Print payload mounts,
    • (2) payloadBrace.stl
    • payloadPlatformLower.stl
    • payloadPlatform.stl
    and mount on axles
  8. Repeat steps 2 to 6 for other set of 4 legs
  9. Mount other leg set on axles.
  10. [optional] Print and mount outside brace bars
  11. Complete payload electronics assembly
